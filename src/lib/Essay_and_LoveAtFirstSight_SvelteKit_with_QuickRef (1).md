[//]: # (Markdown Quick Reference)
[//]: # (- Use # for headings, ## for subheadings, ### for smaller sections)
[//]: # (- Blank lines separate paragraphs)
[//]: # (- **Bold** text uses double asterisks, *italic* text uses single)
[//]: # (- Lists use - or * before each item)
[//]: # (- Quotes start with >)
[//]: # (- Links: [text](url))
[//]: # (- Inline code: `example`; code blocks: ```language ... ```)
[//]: # (- Horizontal line: ---)
[//]: # (- Tables use | pipes and --- dashes)
[//]: # (- In SvelteKit with mdsvex, one # heading at the top becomes the title)
[//]: # (- Lines can be long; VS Code and browsers wrap them automatically)
